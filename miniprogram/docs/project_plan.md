# 项目计划书

## 1. 项目范围
完成微信小程序“智阅·深析”MVP，实现 PDF 阅读、三种角色模式的划词AI解析、笔记系统、基础导出。

## 2. 时间规划（4周）
| 周次 | 任务 | 产出 |
| :--- | :--- | :--- |
| 第1周 | 项目骨架搭建，WebView+pdf.js 渲染上线 | 可翻页查看PDF |
| 第2周 | 划词坐标捕获、高亮回显、笔记锚定 | 闭环的划词-保存-回显 |
| 第3周 | 接入混元大模型，实现 Ben 模式解析 | 划词后返回生活化解释 |
| 第4周 | 增加 David、Alice 模式，基础导出，真机测试 | 提交可审核的小程序 |

## 3. 风险预案
- 若 pdf.js 在 WebView 中性能不佳，退化为 Canvas 自渲染关键页。
- 若混元API调用延迟过高，切换到 DeepSeek 作为主力。

## 4. 质量标准
- 核心流程在 20 篇不同格式论文上测试通过。
- 无阻断性 Bug，界面友好。
