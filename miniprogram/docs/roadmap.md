# 技术路线图

## 已完成

### 核心阅读链路
- PDF 选择与上传（`wx.chooseMessageFile` → 云存储）
- PDF 渲染（HTTP 访问服务 → 云函数 → base64 内嵌 HTML → pdf.js CDN → web-view）
- 分页导航（上一页/下一页）、页码显示

### AI 摘要
- 三角色 Prompt 模板（学生通俗讲解 / 研究者深度分析 / 职场商业洞察）
- 云函数 `pdfSummary` 集成 pdf-parse 文本提取 + DeepSeek API
- 摘要缓存（`documents` 集合，按 userId + fileID 去重）
- 角色切换重新生成

### 基础框架
- 微信授权登录
- 自定义四标签底部导航栏（首页/阅读器/笔记/我的）
- 云开发环境配置（云函数 + 云存储 + 云数据库 + HTTP 访问服务）
- Git 版本管理 + GitHub 远程仓库
